PEQUENO MANUAL DO GIT

- para clonar este repositório
  git clone https://github.com/phpmorales/compiladores.git
  
- para atualizar o repositório, toda vez antes de iniciar as atividades
  git pull
  
- para verificar quais arquivos foram criados/alterados e ainda não foram comitados
  git status
  
- para adicionar antes do commit algum arquivo novo criado
  git add nome_do_arquivo 

- para retirar da lista do 'git status'
  git rm nome_do_arquivo
  
- para limpar a lista de arquivo do 'git status'
  git clean -f
  
- para comitar as alterações realizadas
  git commit -m "alguma mensagem para identificar"
  
- para enviar as alterações para o repositório
  git push origin master
